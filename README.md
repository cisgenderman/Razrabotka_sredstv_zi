# File Encryptor/Decryptor

Консольное приложение для рекурсивного шифрования и дешифрования файлов и папок с использованием AES-256-CBC и библиотеки Crypto++.

## Особенности

- Шифрование/дешифрование всех файлов в указанной директории и её подпапках
- Использование криптографического стандарта AES-256-CBC (Crypto++)
- Паттерн Singleton для централизованного управления шифрованием
- Поддержка русского и английского языков в путях

## Требования

- Windows 7/10/11 (рекомендуется)
- Qt 5.15.2 (MinGW 64-bit)
- Библиотека Crypto++ (для Windows, MinGW-совместимая)
- Компилятор MinGW 8.1.0 (входит в состав Qt)

2. Установите библиотеку Crypto++:

   - Путь к заголовочным файлам и библиотеке должен быть доступен CMake.

## Использование

После запуска приложения откроется консольное меню:
start
1. encrypt
2. decrypt
3. exit
choose:

Параметры:
1 — зашифровать все файлы в выбранной папке и подпапках
2 — расшифровать все файлы в выбранной папке и подпапках
3 — выйти из программы

Пример работы:
Выберите режим 1 (encrypt).
Введите путь к папке, например: C:\Users\user\Documents\
Введите пароль: Password123
Программа рекурсивно зашифрует все файлы, заменяя их зашифрованными версиями.

## Особенности реализации

Алгоритм шифрования: AES-256-CBC (Crypto++).
Генерация ключа: SHA-256 от пароля (QCryptographicHash).
Вектор инициализации (IV): фиксированный
Обработка файлов: Чтение через Qt, шифрование в памяти, запись во временный файл с последующей заменой оригинала.
Обработка ошибок: Исключения Crypto++ перехватываются, временные файлы удаляются.
Логирование: Вывод статуса операций в консоль

## ограничения

Для полного распространения программы необходимо статически слинковать Crypto++ или распространять её DLL вместе с исполняемым файлом.
